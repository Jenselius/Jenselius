## **🚀 GitHub Deployment Instruktioner**

### **Steg 1: Repository Setup**
```bash
git init
git add .
git commit -m "Initial commit: Jenselius Strategy & Ops website"
git branch -M main
git remote add origin https://github.com/[ditt-användarnamn]/jenselius-strategy-ops.git
git push -u origin main
```

### **Steg 2: GitHub Pages Aktivering**
1. Gå till ditt GitHub repository
2. Klicka på **Settings** tab
3. Scrolla ned till **Pages** sektion  
4. Under **Source**, välj **Deploy from a branch**
5. Välj **main** branch och **/ (root)** folder
6. Klicka **Save**

### **Steg 3: Custom Domain (Valfritt)**
1. I **Pages** inställningar, lägg till din **Custom domain**
2. Skapa en **CNAME** fil i root med din domän:
```bash
echo "dindomän.com" > CNAME
git add CNAME
git commit -m "Add custom domain"
git push
```
